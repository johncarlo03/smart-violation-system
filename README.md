# Smart Violation Management System (SVMS) - CTU Danao

## Overview
A web-based administrative tool designed for **Cebu Technological University - Danao Campus** to streamline the monitoring and recording of student violations. This project addresses the need for **Organizational Reliability** and **Administrative Process Efficiency** by digitizing the disciplinary process.

## Key Features
- **Role-Based Access Control:** Distinct interfaces for SAO Administrators, Campus Safety Officers (CSO), and Students.
- **RFID Integration:** Fast student identification using RFID scanning for "Timeliness of Recording."
- **Data Privacy:** Built with the **Data Privacy Act of 2012 (RA 10173)** in mind to secure student records.
- **Automated Notifications:** Email integration for password recovery and violation alerts.

## Tech Stack
- **Framework:** Laravel 12
- **Authentication:** Laravel Breeze
- **Database:** MySQL
- **Frontend:** Tailwind CSS / Blade
- **Testing:** Pest PHP
